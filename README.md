# 🌸🉑HUST自动连接校园网脚本

## 使用场景

当你电脑的校园网可能会被挤掉时，提前打开该脚本（**由于网站可能变更，所以并不保证一定成功**）

## 使用方法

1. 修改login_info文件，第1行为用户名，第2行为密码

2. pip安装selenium

    ```PowerShell
    pip install selenium
    ```

3. 修改auto_connect.ps1中的时间、路径（如python路径）等变量

4. 运行PowerShell脚本

    ```PowerShell
    ./auto_connect.ps1
    ```
